BOT CAIXA MC - INSTRUÇÕES

1. Extraia os arquivos.
2. Renomeie ".env.example" para ".env".
3. Coloque o TOKEN do seu bot no .env.
4. Suba a pasta para o GitHub.
5. No Render:
   Build Command: npm install
   Start Command: node index.js

Comandos esperados no bot:
/caixa
/depositar valor
/retirar valor
/historico

Observação: você ainda precisa registrar os slash commands no Discord Developer Portal ou com um script de deploy.
